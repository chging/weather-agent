# weather-agent
使用ollama实现的最简单的天气查询agent

1.安装ollama
```bash
curl -fsSL https://ollama.com/install.sh | bash
``` 
2.启动ollama
```bash
ollama serve
```
3.下载模型  
```bash
ollama pull qwen3:0.6b
```                 
4.运行程序

安装ollama python包
```bash
pip install ollama
```
运行程序
python3 weather-agent.py
